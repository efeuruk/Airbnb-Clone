# Airbnb-Clone

This repository is a copy of some pages (fully responsive) of Airbnb web page.

I used Webpack, SCSS, Bootstrap, JQuery.

To run, consecutively:

<code>npm i</code>

<code>npm run build</code>

<code>npm start</code>
